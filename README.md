# YouTube Clone Project

This project is a simple **YouTube Clone** built using **HTML, CSS, and JavaScript**.  
It was developed as our **first project** during the **Axsos training program**.

The goal of this project was to practice building real-world web pages, understand layout structure, and collaborate as a team using Git and GitHub.

---

## Project Description

The project simulates the basic structure and layout of YouTube.  
It focuses on frontend development only, without backend functionality or real video streaming.

Each team member was responsible for building a specific page to practice teamwork and task distribution.
